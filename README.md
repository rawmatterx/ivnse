# Intrinsic Value Calculator

A Streamlit app to calculate the intrinsic value of stocks. Ready for local development, Docker, and Heroku deployment.

## Quickstart

```sh
pip install -e .
streamlit run app.py
```

## Architecture

```
ivnse/
├─ ivnse/
│  ├─ data/
│  ├─ models/
│  ├─ ui/
│  ├─ utils/
│  └─ __init__.py
├─ tests/
├─ app.py
├─ requirements.txt
├─ setup.cfg
├─ pyproject.toml
└─ docs/
```
